A estrutura do projeto web api tem alguns arquivos e pastas importantes:

- Dependencies: Dependências do projeto (bibliotecas e ferramentas necessárias)
- Properties: Propriedades do projeto
- Controllers: São classes com métodos que respondem às solicitações dos clientes
- appsettings.json: Configurações do projeto (como configurações de BD, autenticação e autorização)
- Program.cs: Ponto de entrada do projeto.

Códigos de status HTTP usado nos responses (para tratamento de erros)

- 200 OK
- 201 CREATED
- 202 ACCEPTED
- 2024 NO CONTENT
- 304 NOT MODIFIED
- 400 BAD REQUEST
- 401 UNAUTHORIZED
- 403 FORBIDDEN
- 404 NOT FOUND
- 409 CONFLIT
- 500 INTERNAL SERVER ERROR
